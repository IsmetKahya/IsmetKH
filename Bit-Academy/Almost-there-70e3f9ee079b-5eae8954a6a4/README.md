## Klanten
- Azeri is een gebruiksvriendelijke website waar je eenvoudig eten kunt bestellen en laten bezorgen bij jou thuis.

- De website biedt een overzichtelijk menu waarin je zelf je maaltijd kunt samenstellen en specialiseren naar jouw voorkeur. Zo kun je bijvoorbeeld kiezen uit verschillende formaten, sauzen en salades, zodat je bestelling helemaal aansluit bij wat jij lekker vindt.

- Daarnaast ondersteunt Azeri meerdere betaalmogelijkheden, waaronder iDeal, PayPal en andere veilige betaalmethoden, zodat je altijd zorgeloos kunt afrekenen.

- Na het plaatsen van je bestelling ontvang je een bevestiging, en wordt jouw maaltijd zo snel mogelijk bereid en bezorgd.

- Met Azeri wordt eten bestellen niet alleen makkelijk, maar ook persoonlijk en betrouwbaar.

## Admin

- Als admin kun je gebruikmaken van XAMPP. Open XAMPP en start Apache en MySQL. Kopieer de link naar deze map (bijvoorbeeld: http://localhost/Bit-academy/Backend-eindproject/directions/admin.php).

- Nu kun je:

- Het menu bewerken.

- Nieuwe producten toevoegen.

- De bestellingen van je klanten bekijken, inclusief hun gegevens.
